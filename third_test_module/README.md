# Third test module

Кастомный модуль Drupal, который:

1. Добавляет новый тип контента "Проект" .
2. Реализует блок, выводящий 3 последних проекта на главную страницу сайта.

## Установка

```bash
drush en second_test_module -y
drush cr
```
При включении модуля Drupal автоматически импортирует YAML из `config/install/` (тип `project`, поля, form/view display).

Либо через админ-панель Drupal установить модуль в списке модулей. Очистить кэш до и после установки.
**Структура → Макет блоков** → блок **«Последние проекты»** (*Second test module*) → видимость `<front>`.
